## Sobre
Aplicação com diagnósticos e procedimentos de saúde baseados no bando de dados CIAP2 e protocolo SBV. Desenvolvido para auxilio de trabalhadores em programas de treinamentos de empresas.

Tecnologias utilizadas
- `Node.js`
- `Express`
- `EJS`
- `JavaScript`
- `SQLite`
- `Axios`
- `npm`

Pré-requisitos
- `Node.js` (recomendado \>= 14)
- `npm`

Instalação
1. Instale dependências:
   `npm install`

Execução
1. Em modo desenvolvimento:
   `npm run dev`
